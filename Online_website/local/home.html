Connection Settings

We employ Port 465 as the default with the SMTP protocol.

The connection is established using Secure Sockets Layer (SSL) encryption.

How it works

We establish a connection to Gmail through smtp.gmail.com, sending messages using specified accounts. The program employs a parallel spamming method, ensuring the target receives messages simultaneously from multiple accounts.

When using your own Gmail Accounts

Please ensure the entered password is the app password generated by Gmail (16 digits) and not the default password.

Issues and Resolutions

Connection Challenges

If you encounter difficulties connecting to a Gmail account, please ensure the entered password is the app password generated by Gmail (16 digits) and not the default password.

Anti-Virus Detection

Some antivirus software may detect XGhost Spammer as a result of its infinite email sending loops. To resolve this, consider temporarily disabling your antivirus software.

Parallel Spamming Limitations

Kindly note that parallel spamming from different Gmail accounts may not function as expected. This is due to Gmail discontinuing support for less secure apps in May 2022 to enhance account security along with API changes. For more information, refer to discontinuing support for less secure apps
